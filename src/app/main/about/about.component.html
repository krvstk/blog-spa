Serverless SPA with BaaS

Which means this one is a plain single page application with out any real back-end coupled. Angular 12 is used as binding framework. On the other hand I am getting no-sql database, authentication, storage and hosting from Firebase. Sweet combination of both provides fine development and solid scalability.

Here I am using lots of Angular concepts like Reactive Forms, Routing, Animations, Services, Reusable Components, Angular CLI, Lazy Loading and many others. Another exciting and useful nowadays feature is Reactive Programming paradigm. In Angular application it is achieved with RxJs library.

Some solutions of this application may be done in more elegant way, so there is some technical debt I am going to repay in a while. Also I want to dive deeper into NgRx, Angular Animation, Testing, CI/CD, Angular-Material Theming, i18n, enhancement of the project structure, Scss optimization etc. and apply those here. Everything will be described in blog posts.
